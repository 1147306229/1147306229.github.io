  • 截取字符串

    1.使用indexOf获得索引，如果要截取之后的内容还要再加1
    2.使用substring（索引值），即可截取某字符串后面的所有内容

  • 传值

1.前后端互传的类型必须一致
2.如果前台传递的是一个对象，控制层使用（@RequestBody 对象 对象名来接收）
另外，前台使用JSON.stringify（）传递 ，contentType: “appliction/json”

3.注意：如果变量倾斜的话说明是js变量
1.首先，先定义var th = this;
2.th.vue的变量

4.如@click=”showList=false”

  • 测试时间

    long start = System.currentTimeMillis();//这个写在最开始的地方
    long end = System.currentTimeMillis();//写在最后
    System.out.println(Double(end-start)/1000);

  • 获取前年月

    Calendar cal = Calendar.getInstance();
    int year = cal.get(Calendar.YEAR);
    int month = cal.get(Calendar.MONTH) + 1;

  • shiro拦截器

    拦截器名 说明

    anon 匿名拦截器，即不需要登录即可访问；

                        一般用于静态资源过滤；示例“/static/
                    **=anon”

  • 单选框和多选框

    1.都是要带name，各自对应一个name，如果有多个题目则不同，value也不同
    2.想要拿到值，可以使用@change或者click事件

  • shiro

    1.shiro的核心API
    Subject：用户主体（把操作交给SecurityManager）
    SecurityManager：安全管理器（关联Realm）
    Realm：shiro连接数据的桥梁

    2.依赖
    org.apache.shiro
    shiro-spring
    1.4.0

    3.属性salt：加盐
    source：密码
    AlgorithmName：加密算法名称
    hashIterations：加密算法次数

  • thymeleaf

    1.thymeleaf3.0以前对页面标签语法比较严格，开始标签必须有对应的标签

  • idea快捷键

    1.局部变量变成员变量快捷键：ctrl+alt+f
    2.ctrl+h 查找实现类
    3.ctrl+alt+o 清除错误的包
    4.shift+shift查找项目中所有的东西

  • springboot热部署

    1.首先在idea的File-settings里找到Compiler把Build
    project automatically（自动编译）打上勾。

    2.但只打开自动编译，运行时时不生效的。所以要按 ctrl+shift+alt+/
    选择registry，找到下图选项(compiler.automake.all….)并打上勾
    才能在运行时自动编译。

    3.在pom.xml文件中添devtools依赖。
    devtools实现对class文件变化的监听，以及页面的热部署。

    org.springframework.boot spring-boot-devtools 2.2.2.RELEASE 4.在application.properties配置文件关闭缓存，使页面能够实时刷新 #关闭缓存功能 spring.thymeleaf.cache=false

  • ajax

    使用ajax时如果只是传递单个字段可以使用async: false,
    如果把这个选项设置成false，这意味着所有的请求都不再是异步的了
    如果后台用的是对象接收是接收不到的

    注意：要拿ajax里面的值在ajax方法里拿

  • QueryWrapper

    1.在实现类中的queryPage方法中多个条件构造器，也就是
    IPage page = this.page(
    new Query().getPage(params),
    new QueryWrapper()
    .eq(“这是要查询数据的数据库的主键字段”,id);

    或者这样写，写在for循环中
    QueryWrapper queryWrapper=new QueryWrapper();
    queryWrapper.eq(“user_id”, scoreEntity.getUsId());
    );

    getById和getOne
    使用getOne里面的值需填写queryWrapper，而getById填写对象点属性

  • underfined

    1.Cannot read property ‘id’ of undefined类似于这种js点不出，数组越界问题

    2.没有定义这个属性

  • v-mode对象绑定

    1.在js中声明一个变量如list{},然后在html页面只需在input上面用
    v-model=’list.属性名”，这个属性名必须是对象中存在的，这样的话
    在插入方法就可以实现的很简单了
